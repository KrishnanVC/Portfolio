Facial Gestures based PC control
In this project we used OpenCV to detect the facial gestures of a person and use that to control the PC. We used the persons nose as an reference to move the cursor around the screen and wink to specify a click. We added a lot of usability features that makes using this app easy. This project was built with the intention to help the differently abled.

Energy Auditing using Machine Learning
Leveraging the power of machine learning, to predict the electricity bill for the current month in order to limit the usage, before it goes beyond the threshold. This project was built to help deal with the global energy crisis by making the user to use the electricity sensible. It also benefits the user as they have to pay less.
